# core

## 核心功能
zero-core 的核心逻辑层，提供配置管理、系统提示词构建、工具策略、上下文管理、Hook 注册、日志、压缩（compaction）等基础能力，是所有上层模块的共享基础设施。

## 输入
用户配置文件、Agent 定义、工具注册请求

## 输出
配置校验结果、系统提示词、工具策略决策、Hook 调度、日志输出

## 定位
src/core/ — 被所有其他模块依赖的共享基础设施层，无内部模块依赖

## 依赖
外部依赖：zod（schema 校验）等；无内部模块依赖（被所有其他模块依赖）

## 维护规则
- 核心模块，修改需谨慎评估对全局的影响
- 新增配置项需在 config.ts 的 schema 中注册
- 新增 Hook 类型需在 hook-types.ts 中定义
- 常量变更需检查所有引用处
